# :material-playlist-check:{ .md .middle } Warp Pro 订阅

![Warp Pro 订阅](../images/warp-pro-sub.jpg)

Xray、Xray Knocker 与 Mihomo 核心在 GFW-Knocker 的改进下实现了 Warp 解锁，催生了 **MahsaNG**、**v2rayN-PRO** 等应用。这些改进针对伊朗网络环境优化 Warp 连接，类似 **Oblivion** 团队的工作。面板新增 **Warp Pro** 订阅，并提供可调参数。

各 ISP 的最佳值需实测确定，可能随时间变化，类似分片设置。默认值已验证可用，你只需要扫描并设置合适的端点即可。

可下载 Warp Pro 配置的 zip 供 **WG Tunnel** 使用。客户端中点击 `+` 导入 zip 即可，无需额外设置。**Amnezia** 也类似，但其对 zip 导入不稳定，请先解压，再逐个导入所需配置并连接。
