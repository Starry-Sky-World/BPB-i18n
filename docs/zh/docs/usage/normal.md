# :material-playlist-check:{ .md .middle } 常规订阅

![常规订阅](../images/normal-sub.jpg)

此订阅会应用最优的 VLESS/Trojan 设置、DNS、路由规则等，以减少用户配置失误，并包含 **Best Ping** 配置。

???+ question "什么是 **Best Ping** 配置？"
    该配置会汇总面板中的所有配置，每 30 秒测试一次以找到并连接最快的配置。如果你添加了 Clean IP、启用 Trojan 或选择了额外端口，这些也会包含在 **Best Ping** 中。此功能在 Fragment 与 Warp 订阅中同样提供。

应用路由规则后，该订阅可屏蔽约 90% 的伊朗与海外广告，直连伊朗/中国/俄罗斯网站（无需关闭 VPN 访问支付网关），绕过局域网，屏蔽色情内容与 QUIC 协议；对于 Sing-box 与 Clash 订阅，还能有效屏蔽钓鱼与恶意软件内容。
